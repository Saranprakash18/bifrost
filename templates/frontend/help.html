Get started with transforming your UI designs into clean frontend code

Dashboard Overview

1

Default Templates

Your dashboard includes several default templates you can use immediately. Click on any template to see the generated code.

2

Upload New Design

Click the "Upload" button to convert your own UI screenshots into code.

Uploading Your Design

1

Prepare Your Screenshot

Capture a clear screenshot of your UI design. For best results, make sure the screenshot is high-quality and all elements are visible.

2

Upload the File

Click the "Upload" button in your dashboard, select your screenshot file, and wait for Bifrost to process it.

3

View Results

After processing, you'll be redirected to the results page with your generated HTML, CSS, and JavaScript code.

Working with Generated Code

1

Code Preview

The results page shows your code in organized tabs for HTML, CSS, and JavaScript. You can switch between them to view each part.

2

Copy Code

Click the "Copy" button next to each code section to copy it to your clipboard for use in your project.

3

Download Code

Use the "Download" button to save the complete code as a ZIP file containing all necessary files.
